🌍 旅行攻略 API

移动软件测试工具体验作业 - 后端服务

服务运行中

👤 用户接口

  • POST /api/users/register 用户注册
  • POST /api/users/login 用户登录
  • POST /api/users/logout 用户登出
  • GET /api/users/me 获取当前用户

📍 目的地接口

  • GET /api/destinations 获取所有目的地
  • GET /api/destinations/{id} 获取目的地详情
  • GET /api/destinations/search?location=xxx 搜索目的地
  • GET /api/destinations/popular 获取热门目的地

📝 攻略接口

  • GET /api/guides 获取所有攻略
  • GET /api/guides/{id} 获取攻略详情
  • GET /api/guides/search?keyword=xxx 搜索攻略
  • POST /api/guides/{id}/like 点赞攻略

📊 用户行为追踪接口

  • POST /api/behavior/page-view 记录页面访问
  • GET /api/behavior/session/{sessionId} 获取会话行为记录
  • GET /api/behavior/user/{userId} 获取用户行为统计

⚠️ 异常测试接口（供Fasttestsuit捕获）

  • GET /api/test/null-pointer?trigger=true 空指针异常
  • GET /api/test/arithmetic?divisor=0 算术异常(除零)
  • GET /api/test/index-out-of-bounds?index=10 数组越界异常
  • GET /api/test/number-format?number=abc 数字格式异常
  • GET /api/test/resource-not-found?id=999 资源未找到异常
